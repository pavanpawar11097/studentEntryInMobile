A new Flutter project.

## Getting Started

This project demonstrates the implementation of a **local database system in Flutter** using SQLite (`sqflite`) and `path_provider`.  
It is designed as a learning resource for students or developers to understand **CRUD operations**, local data persistence, and Flutter UI integration.

Key learning objectives:

- Understanding **SQLite database integration** in Flutter applications  
- Implementing **Create, Read, Update, Delete (CRUD)** operations  
- Using **dialogs** (`AlertDialog`) for data entry and editing  
- Managing **state updates** and refreshing UI after database operations  
- Handling **platform-specific file paths** for storing data locally
